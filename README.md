# Crypto IoT Protocols Demo

> **Video demo:** []

This repository contains several TypeScript demos developed for an IoT security assignment.  
The project demonstrates cryptographic mechanisms and lightweight communication protocols applied to constrained-device scenarios.

The repository includes:

- A homomorphic encryption aggregation demo
- A blind signature demo
- A CoAP / CoAP-to-HTTP reverse proxy integration
- Additional energy-related IoT demo files
